## Run manually

### Run dev
```bash
bun run dev
```

### Build
```bash
bun run build
```

### Run prod
```bash
bun run start
```

### Use

```bash
curl localhost:3000`
```

OR

Open http://localhost:3000/ with your browser to see the result.
